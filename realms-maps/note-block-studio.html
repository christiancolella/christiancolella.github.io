Note Block Studio

Note Block Studio is a map in which the player(s) can sequence note block music with a friendly and interactive user interface. Players can create 64 measures of music on a 73-key piano and a 3-piece drum kit. They are also able to save and load up to 5 of their songs and choose between 3 tempo settings.
